简介

通常为了允许用户最终访问Web应用程序，我们会把代码部署到云服务器或云托管平台上。虽然有许多工具可以实现这一点，但在本文中，我们将介绍如何将 FastAPI 应用程序部署到 Vercel。

FastAPI 是一个现代且快速的 Python Web 框架，用于构建后端 API 应用程序。FastAPI 支持由 Swagger、安全模块和类型检查提供支持的 API 文档，以确保代码的正确性。
